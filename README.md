# studentattendancemanagementsystem
📌 Student Attendance Management System
📖 Overview

The Student Attendance Management System is a front-end web application designed to manage and track student attendance efficiently. It includes role-based dashboards for Admin, Teacher, and Student, making it suitable for educational institutions like schools, colleges, and coaching centers.

The project is built using only HTML, CSS, and JavaScript, with localStorage used for data storage.

🛠 Tech Stack
HTML5
CSS3 (Glassmorphism UI Design)
JavaScript (Vanilla JS)
localStorage (for data persistence)
🚀 Features
🔐 Login System
Role-based login (Admin / Teacher / Student)
Simple authentication using localStorage
👨‍💼 Admin Panel
Manage students and teachers
View attendance records
Dashboard analytics overview
👨‍🏫 Teacher Panel
Mark attendance (Present / Absent)
View and update attendance records
Filter data by date
🎓 Student Panel
View personal attendance
Check attendance percentage
View history of attendance
📊 System Features
Attendance tracking system
Real-time statistics
JavaScript-based charts (no libraries)
Dark mode / Light mode toggle
Fully responsive design
⚠️ Validation Rules
Name: Only alphabets allowed
Age: Only numbers (5–100 range)
Email: Must be valid format
Phone: Exactly 10 digits
💾 Data Storage

All data is stored locally in the browser using localStorage, so no backend or database is required.
